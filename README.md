## 搬瓦工是什么？定义先说清楚

## 三档价格背后的逻辑

### 一、入门档：$49.99/年

这个价格买到的是 CN2 套餐或 KVM 套餐，两个叫法，配置基本一样。

线路走的是 CN2 GT（全球中转），比最普通的国际线路强，但没有 CN2 GIA 那么稳。白天用问题不大，晚高峰可能会感受到一点延迟抖动。

适合刚接触 VPS、想学 Linux 或者跑个小博客的人。不对速度有苛刻要求的话，这个价格入手完全够用。

说实话，我自己最开始就是这个价位进来的。拿来跑了半年 Nginx + WordPress，稳定性没什么问题，但后来业务量上来了才发现晚高峰确实偶尔卡顿，那时候才升上去。

### 二、主力档：CN2 GIA-E，$169.99/年起

这是搬瓦工用得最多的套餐系列，"性价比最高"这个词被提到频率最高的一个档位。

线路质量和入门款是两个级别。CN2 GIA-E 走的是中国电信的 CN2 GIA 专线（ECOMMERCE 版，带宽更大），加上三网优化——电信走 CN2 GIA，移动接入 CMIN2，联通走高端路由。晚高峰表现稳。

另一个优势是机房切换。买完之后，12 个机房可以随时在 KiwiVM 后台一键迁移，包括洛杉矶 DC6、DC9、日本大阪软银、荷兰联通 EUNL_9 等，不加钱、几分钟搞定。哪个机房你用着不舒服，直接迁走就行，零成本试错。

带宽也升到了 2.5Gbps，比入门款的 1Gbps 翻了一倍多。

不知道选什么的话，直接选这档，闭眼买。

👉 [以 $49.99/季 开始使用 CN2 GIA-E 套餐](https://bit.ly/BanWaGon)

### 三、高端档：香港套餐，$89.99/月起

香港套餐最少 $89.99/月，一年下来将近 $900。买的人确实少。

但它解决的是一个特定问题：极致低延迟。香港机房到国内大部分地区的延迟在 30-60ms 之间，广东方向甚至能到个位数毫秒，这是地理位置决定的，不是线路技术能弥补的差距。

如果你跑实时交互类应用、对延迟有极高要求，或者是公司报销，香港套餐值。普通建站或学习用途，CN2 GIA-E 已经够了。
